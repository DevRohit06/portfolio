# Product Overview

This is Rohit Kushwaha's personal portfolio website - a modern, full-stack developer portfolio showcasing projects, blog posts, and professional experience.

## Key Features

- Personal branding and professional showcase
- Interactive experience timeline with smooth animations
- Blog system with MDX content
- GitHub activity integration showing live contributions
- Discord status integration
- Dark/light mode theming
- SEO optimized with structured data
- Responsive design for all devices

## Target Audience

- Potential employers and clients
- Fellow developers and tech community
- Blog readers interested in web development and AI topics

## Brand Voice

- Casual, friendly, and approachable ("Hey, I'm Rohit")
- Tech-savvy but accessible
- Enthusiastic about AI and modern development
- Personal and authentic storytelling
